# Contributions to Helctic are complicated

### in order for you to contribute:
 - You must have initmate knowledge of the Rust Programming Language
 - You must have a knowledge of the way that Drivers and SDK works
 - You must either be approved by an Helctic or a [Helctic](https://Helctic.dev) Core Dev

 ## Most of the Discussion happens in our [Discord](/community/discord) You can read more on the linked page
